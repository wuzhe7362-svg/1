案例四：河南省新乡市世富房地产开发有限公司与河南省广厦建设工程有限公司建设工程施工合同纠纷申请再审案，(2015)民申字第2169号，《人民司法·案例》2016年第5期。
【基本案情】
再审申请人世富公司为与广厦公司建设工程施工合同纠纷一案，不服河南省高级人民法院（2013）豫法民一终字第76号民事判决，向最高人民法院申请再审称：一审、—审判决文书所依据的最主要的证据是豫公专[2012]痕鉴字第0072号河南公专司法鉴定中心关于新乡市世富房地产开发有限公司印章检验鉴定书。2014年11月11日，该中心以关于撤销豫公专痕鉴字0072号鉴定书决定（以下简称撤销决定）文件，撤销了原鉴定书。该撤销决定是证明人在一审、二审中向法庭提交的2010年11月30日证明是否有效的关键证据，属新证据。
　河南公专司法鉴定中心称：撤销决定是经世富公司申请作出的。世富公司要求鉴定中心撤销原鉴定的依据是司法部《司法鉴定执业分类规定》（试行）（以下简称《分类规定》）第三条：“本执业分类规定是确定面向社会服务的司法鉴定人职业（执业）资格和司法鉴定机构鉴定业务范围的依据。”第十条：“文书司法鉴定：运用文件检验学的原理和技术，对文书的笔迹、印章、印文、文书的制作及工具、文书形成时间等问题进行鉴定。”第十一条：“痕迹司法鉴定：运用痕迹学原理和技术，对有关人体、物体形成痕迹的同一性及分离痕迹与原整体相关性等问题进行鉴定。运用枪械学、弹药学、弹道学的理论和技术，对枪弹及射击后残留物、残留物形成的痕迹、自制枪支和弹药及杀伤力进行鉴定。”据此，世富公司称本案证明中所盖世富公司印章真实与否应属文书司法鉴定，而非原鉴定中的痕迹司法鉴定。由于鉴定人李西亭、詹绍旭在作出原鉴定时的执业范围不包括文书司法鉴定，故应撤销原鉴定书。但司法鉴定中心又称：印章印文既有文字属性，更有痕迹属性，还有图像的属性。多数印章的特征痕迹属性更突出，现代技术手段更是倾向于将印章鉴定用痕迹手段处理，更科学，更相邻。传统的公安业务中，刑事技术鉴定人员综合运用痕迹技术、印文印章鉴定技术和文检鉴定技术解决印文印章鉴定问题实为正常。鉴定中心指派的两名鉴定人为河南警察学院教授和副教授，二人长期从事刑事技术、刑事侦查教学和科研工作，具有痕迹、图像的鉴定能力，能够着实解决本案鉴定问题，没有错误。
